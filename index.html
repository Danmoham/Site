    Tick Tock Task

    Presenting Tick Tock Task, my meticulously crafted full-stack web application. Designed for both mobile devices and desktop devices, this app has been created to streamline event tracking and enhance time management skills. Utilizing a cutting-edge generative AI API, the app provides approximate timeframes for specific activities. Users can effortlessly manage their schedule through an intuitive calendar interface, with the ability to add or delete events exclusive to the current date. Both a hosted link and the github repo is available to checkout below.

    PlayPal

    PlayPal is a web application made by myself and a team of four others that brings gamers together based on the games they share a common interest for. The app has access to over 200 000 games using the rawg API to fetch this data and reviews can be left for each game, you are suggested friends that you can make based on the review scores you give which can allow you to add friends, make group chats and add events to your gaming calendar with our calendar feature. To see a full demonstration of this in action, please click our video demo link.

    Nc News Web Application

    Nc News has been developed to emulate a real world news web application that allows users to interact with the news site by reading articles, commenting on articles, sorting articles and even posting their own articles! The App has been developed to focus on user experience using optimistic rendering when necessary. If you wish to give the site a go for yourself, please click the hosted site button below (please note this uses a third party hosting platform so there is a delay in getting data from the api).

    Nc News Api

    This Api was developed using PSQL and JS running with a node environment using testing with Jest. This was developed for the back-end component for NC news to store all the data required for users to interact with the news outlet. This has multiple end points that allow users to patch, post and delete from different tables in the database. Github repo and hosted site links below.
